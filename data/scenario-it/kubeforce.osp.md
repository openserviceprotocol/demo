# KubeForce Systems

> Kubernetes-native cloud infrastructure. We operate container
> platforms on AWS, Azure, and Google Cloud — the only managed
> Kubernetes provider in the DACH region covering all three
> hyperscalers. For teams that build cloud-native and don't want
> to manage the platform. Based in Vienna.

## Available Services

- [Managed Kubernetes](/osp/services/managed-k8s.yaml): Kubernetes clusters on AWS/Azure/GCP, fully managed, multi-cloud capable
- [Platform Engineering](/osp/services/platform-engineering.yaml): Building an internal developer platform on Kubernetes
- [Cloud Migration to Kubernetes](/osp/services/k8s-migration.yaml): Migration from VMs or Docker Compose to Kubernetes

## Not Available

- Classical VM hosting without containers
- Managed databases (we operate the platform, not the data layer)
- Windows workloads
- On-premise Kubernetes (cloud only)
- Security audits (we refer to specialized partners)
